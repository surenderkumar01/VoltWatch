VoltWatch

VoltWatch is an Android app that shows battery information and saves battery history.

Features
Shows battery level, temperature, voltage, and technology
UI color changes based on battery level
Saves battery data every 15 minutes
View battery history
Alert when battery reaches selected percentage
Works even when app is closed
 Architecture

This app uses MVVM architecture.

Flow:
Battery → BroadcastReceiver → ViewModel → Repository → Room Database → UI

Tech Used
Kotlin
Jetpack Compose
MVVM
Room Database
Hilt
Coroutines & Flow
WorkManager

 Screens
Dashboard (current battery data)
History (saved data)

 How to Run
Clone the project
Open in Android Studio
Run on emulator or device
